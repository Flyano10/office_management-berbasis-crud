# PLN Color Palette

## Warna PLN - Teal Palette

### Primary Colors
- **PLN Primary Teal**: `#4A9B9B` - Warna utama PLN
- **PLN Dark Teal**: `#2D5A87` - Warna gelap untuk kontras
- **PLN Light Teal**: `#6BB6B6` - Warna terang untuk aksen
- **PLN Lighter Teal**: `#8CC8C8` - Warna paling terang untuk highlight

### Usage Guidelines
- **Primary Teal (#4A9B9B)**: Digunakan untuk button, icon, dan elemen utama
- **Dark Teal (#2D5A87)**: Digunakan untuk background gelap dan kontras
- **Light Teal (#6BB6B6)**: Digunakan untuk gradient dan aksen
- **Lighter Teal (#8CC8C8)**: Digunakan untuk highlight dan detail

### Implementation
Warna ini telah diterapkan di:
- Blue Banner (Top Left)
- Abstract Shapes (Blue Curve)
- Wave Elements (Wave-1, Wave-2, Wave-3)
- CSS Variables (--pln-accent)
- Button Colors
- Icon Colors
- Scrollbar Colors
- Background Gradients
